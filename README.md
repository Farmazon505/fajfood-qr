# QR на стол

Гостевой QR-сервис: вызов официанта с причиной, Telegram-уведомления, админ-панель, печать QR-кодов и карта лояльности, синхронизированная с CRM и iiko.

## Запуск

```bash
npm install
copy .env.example .env
npm run dev
```

Гостевая страница: `http://localhost:5174/t/table-1`

Админка: `http://localhost:5174/admin`

Пароль берется из `ADMIN_PASSWORD` в `.env`. Если `.env` не создан, временный пароль: `admin123`.

## Telegram

1. Создайте бота через BotFather.
2. Запишите токен в `TELEGRAM_BOT_TOKEN`.
3. Для локальной разработки поставьте `TELEGRAM_ENABLE_POLLING=true`.
4. В админке добавьте официанта и укажите его `telegramChatId`.

Если токен не задан, вызовы сохраняются в системе, но уведомления только логируются в консоль.

## Лояльность

`Qrnastol` не хранит ключ iiko. Регистрация отправляется по серверному каналу в CRM:

```env
CRM_BASE_URL=http://127.0.0.1:3000
CRM_LOYALTY_SERVICE_SECRET=shared-random-secret-with-at-least-32-characters
```

В CRM должен быть тот же `CRM_LOYALTY_SERVICE_SECRET`, а также:

```env
LOYALTY_WELCOME_BONUS_AMOUNT=500
IIKO_LOYALTY_WALLET_ID=
```

При регистрации система отдельно фиксирует обязательное согласие на обработку персональных данных и необязательное согласие на рекламные сообщения. Для доказательства сохраняются версия и SHA-256 текста, дата, IP и User-Agent.

Номер подтверждается бесплатно через контакт текущего пользователя в Telegram или MAX. Сайт не принимает номер из мессенджера на доверии: CRM проверяет Telegram `user_id` либо подпись контакта MAX, срок действия ссылки и совпадение номера с анкетой.

Для безопасного тестового запуска укажите разрешенные номера через запятую:

```env
LOYALTY_REGISTRATION_ALLOWLIST=+79990000000,+79990000001
```

## Что уже есть

- QR-страница столика `/t/:slug`.
- Настраиваемые кнопки вызова: счет, заказ, помощь, кальянщик и любые другие.
- Telegram-сообщения официантам с кнопками "Принял" и "Готово".
- Админка для заведения, акций, кнопок, столов, официантов и заявок лояльности.
- Печатный лист QR-кодов с короткой инструкцией для гостей.
